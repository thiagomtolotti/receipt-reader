# Receipt Reader

This project is a receipt reader that receives an image of a receipt and returns its content in a structured format. The application will use an AI model to extract relevant information from the receipt (store name, date, total amount, items, etc.) and present it in a user-friendly format. The goal is to create a tool that can help users easily manage their expenses by digitizing their receipts.

## Technologies

### Backend

- Python
- FastAPI
- AI model (to be determined)
- SQLite/PostgreSQL (for storing receipt data)

### Frontend

- React
- Vite
- Tailwind CSS
- Shadcn UI
